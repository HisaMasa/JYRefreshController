JYRefreshController
===================

An easy way to add pull-to-refresh or pull-to-loadMore to any kind of UIScrollView.

## Demo

<img width="200" src="http://alvin-blog.qiniudn.com/jy_refresh_controller_demo.gif">

## Podfile

```ruby
platform :ios, '6.0'
pod "JYRefreshController"
```
